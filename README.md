# MSA Wits Website

Official website for the Muslim Student Association (MSA) at Wits University.

## 🌙 About

This platform serves as the central hub for the MSA, providing students with access to events, prayer information, and community resources on campus.

## Tech Stack

- Next.js
- Tailwind CSS
- Vercel (deployment)

## Features (Planned)

- Upcoming events (halaqahs, socials, Jumu'ah)
- Prayer & Jumu'ah information
- Committee/team page
- Contact & join form
- Islamic resources

## 🛠️ Getting Started

Clone the repository:

git clone https://github.com/Amaan-Hans/msa-wits.git
cd msa-wits

Install dependencies:

npm install

Run the development server:

npm run dev

Then open:

http://localhost:3000

## 📦 Deployment

This project is deployed using Vercel.
